1. innerText VS. innerHTML

innerText

innerText is property can be applied to DOM objects in Javascript to be able to get and edit text content only of the element (without any tags); That means if you put HTML code with innerText it will be shown as the code you put like "<p>content</p>".

innerHTML

innerHTML is another property for DOM objects same as innerText property but you can get and edit HTML tags.

2. array built-in functions

Open console to see the effect of the written script.

map(callbackfn)

map function returns new array that contains edited values of the array. map function accept one parameter that should be a callback function. the callback function should has two parameters value and index. the callback function should return the new value of the current index. value parameter is the value of the current index. index parameter is telling us where we are in the array.

filter(predicate)

filter function returns new filtered array. filter function accept one parameter that should be a function. the function should has two parameters value and index. the function should return boolean value; If the boolean value is true, the function keep the current value and if it's false it removes the value from the array.

reduce(callbackfn)

reduce function just like map function but it accept two parameters: function with three parameters. previousvalue: it contains the value of the last iteration, currentvalue: it contains the value of the current index, currentindex. the second parameter of reduce function is the default value if there is no previous value.
